### Specifications
- Webpack
- Eslint airbnb config
- Code splitting, gzipping
- Dot env  
- React, React-router
- Markdown parser

### Installation
- Clone this repo into a local folder.
- Run `npm run install`.
- Run `npm run start` for development and hot reloading.
